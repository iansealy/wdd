WDD11 Reading

After the WDD11 session, you may like to look at some of these resources.

If you're not very good at HTML then you must read these tutorials.

  • Getting started with HTML
  • HTML School

There's really no need to spend money buying books about learning HTML because all the information you need is available online. But if you work offline a lot and need a good HTML reference book then I recommend:

  • HTML for the World Wide Web with XHTML and CSS

If you desperately want a book about learning HTML then I recommend:

  • Sams Teach Yourself HTML & XHTML in 24 Hours

The definitive reference for what's allowed in HTML 4 (the current version of HTML, kind of) is:

  • HTML 4.01 Specification

It's not what you'd call light reading though.
