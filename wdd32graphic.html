Jonathan Fox

Designing Web pages in Photoshop

Designing Web pages in Photoshop has its disadvantages:

  1. If you're not familiar with Photoshop you need to invest time in learning it
  2. Photoshop will only produce a static image of a page and this may confuse clients
  3. Pages look will different in different browsers

Designing Web pages in Photoshop

However:

  1. It's not only about presenting to clients
  2. It's much easier to be creative using Photoshop
  3. You can experiment with colour, type and layout without having to translate every idea into code
  4. You can lift all of the graphic elements out of your mockup and use them in a working version

Designing Web pages in Photoshop

Conclusion:

  • It comes down to personal choice

Colour

Using colour

  • Colour is probably the first thing you'll notice on a Web page
  • Colour is an effective way to:
    • attract attention
    • group elements
    • create visual interest
    • enhance aesthetics
  • Different colours mean different things to different people

Colour Schemes

Using colour (continued)

  • When you design for print every colour adds a cost to the job
  • On the Web you've got 16 million colours to pick from and they're all free
  • There are tried and tested colour schemes or ways of combining colours

Colour Scheme tools

Online tools that will help you pick a colour scheme for your site:

  • Colour Scheme Designer
  • Kuler (requires Flash)

Typography

When you're designing a Web page in Photoshop you can add text as:

  1. Body text (or live text) that is content
  2. Image

Typography

When you're adding text that is content to your mockup:

  1. Use Web Safe fonts
  2. For text below 12 pixels turn anti-aliasing off as this will give a better indication of what it will look like in a browser

Typography

In some situations you may need to use image as text:

  • You want to use a non-Web Safe font
  • You want to add text to an image
  • You want to set text in particular way
  • You want to apply a particular effect to some text
