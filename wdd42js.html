What Is JavaScript?

  • A scripting language
  • Mainly used on the client rather than the server
  • Allows interactivity without using CGI scripts or similar technologies
  • Because there is no connection to the server, JavaScript can be fast and therefore allows instant interactivity (e.g. respond to mouse clicks or movement)
  • Biz/ed Question Bank & Gmail
  • Example uses of JavaScript

Java And JavaScript

  • JavaScript is a relatively simple scripting language used on the Web
  • Java is a complex programming language used for a wide range of applications (which are not necessarily Web-related)
  • Java and JavaScript are completely different

Where Is JavaScript?

  • Embedded within HTML
  • Usually in the HEAD of a document
  • Javascript starts with
    <script language="JavaScript" type="text/javascript">
    and ends with
    </script>
  • JavaScript is often flanked by
    <!--
    and
    // -->
    to prevent it being displayed by older browsers
  • Can be embedded in an HTML tag

Problems With JavaScript

  • Like CSS, JavaScript does not always work the same in all browsers
  • Because of all the different versions, it is sometimes still necessary to detect browser type

Netscape History

  • JavaScript was developed by Netscape for Navigator, but was then called LiveScript
  • Was renamed JavaScript for marketing reasons (even though very different to Java)
  • Navigator 2.0 (1996) supports JavaScript 1.0
  • Navigator 3.0 (1996) supports JavaScript 1.1

Netscape History

  • Navigator 4.0-4.05 (1997) support JavaScript 1.2
  • Navigator 4.06+ (1998) support JavaScript 1.3
  • Netscape 6.0+ (2000) and Mozilla support JavaScript 1.5
  • Firefox 1.5 (2005) supports JavaScript 1.6
  • Firefox 2 (2006) supports JavaScript 1.7
  • (JavaScript 1.4 was used in a Netscape server product)

Microsoft History

  • Microsoft created its own version of JavaScript called JScript (Netscape owns the name)
  • IE 3.0 supports JScript 1.0 (1996) and JScript 2.0 (1997), but is very different to JavaScript
  • IE 4.0 (1997) supports JScript 3.0

Microsoft History

  • IE 5.0 (1999) supports JScript 5.0
  • IE 5.5 (2000) supports JScript 5.5
  • IE 6.0 (2001) supports JScript 5.6
  • IE 7.0 (2006) supports JScript 5.7
  • Confusingly IE3.0+ also support a scripting language called VBScript, which is not supported by other browsers
  • (JScript 4.0 was used in another Microsoft product)

Standards History

  • Netscape, Microsoft and others collaborated with ECMA to create an official JavaScript specification called ECMAScript
  • ECMAScript v1 (or ECMA-262) is most similar to JavaScript 1.3

Standards History

  • Navigator 4.06+ (JavaScript 1.3) and IE 4+ (JScript 3.0+) are ECMAScript-compliant
  • Therefore the Web's "official" scripting language is based on Netscape's JavaScript
  • But the Web's "official" Document Object Model is based on Microsoft's
  • ECMAScript v2 was only released for maintenance purposes
  • ECMAScript v3 is like JavaScript 1.5 or JScript 5.5
  • ECMAScript v4 is a work in progress and will be (exactly) like JavaScript 2.0
