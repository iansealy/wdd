Why Link Checking Is Important

  • In an ideal world, URLs would never change (see Cool URIs don't change and URL as UI)
  • Unfortunately, links do break - 5.7% of the links surveyed on 200 pages in May 1999 were broken (and 28.5% of those pages contained broken links)
  • Broken links make your site look unprofessional and reduces user confidence
  • Especially broken internal links
  • Therefore all sites should be link checked before being made public (and regularly after go live)

How To Link Check

  • Link Valet
  • Link check (plus extras) 5 pages using NetMechanic

Common Error Codes (HTTP Status Codes)

  • 404 - Not Found
    Permanent. Link is broken.
  • 503 - Service Unavailable
    Perhaps temporary. The Web site may be undergoing maintenance or there may be a network problem.
  • 500 - Internal Server Error
    Perhaps temporary. The Web server has found an error whilst trying to display the page.
  • 401 - Unauthorized
    Access requires a username and password.
  • 403 - Forbidden
    Access is not allowed (e.g. because access is restricted to specific machines).
