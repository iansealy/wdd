Types of Layout

Three main types of layout:

  • Fixed width
  • Fluid width (aka liquid width)
  • Elastic width

Basic HTML

Create basic HTML for all layouts. For this, I've used:

  • W3C template and DOCTYPE
  • YUI Reset CSS
  • Container, header, footer and three columns
  • Lorem ipsum placeholder text
  • Basic CSS with text sized in ems and colours chosen with Color Scheme Designer
  • Menus from Listamatic

Fixed width

  • Easiest to develop
  • Can make sure content is optimal reading width (unless user changes font size)
  • Example

Fluid width

  • Relatively easy to develop
  • Example
  • User has control of optimal reading width (but will they)
  • Example

Elastic width

  • Hardest to develop
  • Width adapts to font size
  • New browsers support page zoom
  • Example

CSS3

  • Modular extension of CSS2.1
  • Some modules are finished and have wide support; others do not
  • CSS Validation Service now defaults to CSS3
  • e.g. border-radius and selectors
  • CSS3.info
