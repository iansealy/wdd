Welcome to the Web site for the Birkbeck Postgraduate Certificate in Web Design and Development.

This site is intended to support you during the course. It contains details of the staff teaching the course, as well as some information about all the students. More importantly, the modules section contains full notes for each session. There is also information about mailing lists, details of each of the four assignments, a timetable and a reading list.
