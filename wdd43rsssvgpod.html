RSS News feeds

RSS is an XML format designed for sharing news headlines or "What's new?" for a particular Web site.

Essentially you provide an RSS feed which is updated as and when something new appears on your Web site. Other people pick up the feed and present a list of current headlines on their site - these headlines link back to your original articles.

So, how do you create your own RSS feeds?

  • By hand - time consuming and you're liable to introduce errors.
  • Using a program on your server that queries a database and produces the RSS for you - requires technical knowledge and ability to run scripts and databases on your server.
  • The feed might be already provided if you use a third-party blogger/news group.

Podcasting

What is podcasting

Podcasting is a method of publishing audio broadcasts (usually mp3 files) via the Internet. It allows users to subscribe so they automatically receive new broadcasts as they are published.

These audio broadcasts are called podcasts as they were originally intended for use on iPods. However, podcasts can be downloaded and listened to on a variety of devices: PCs, mobile phones, PDAs, mp3 players, as well as iPods.

Similar to RSS but uses <enclosure> tag.

SVG

What is SVG?

Scalable Vector Graphics

  • It is a vector-based graphics format
  • XML application
  • Open format (unlike Flash)
  • It's "readable" - view source
  • Huge accessibility advantages
  • Can be animated
  • Uses CSS

What is SVG? continued

  • You can add metadata to an image
  • Small filesize
  • W3C recommendation
  • Interoperability
  • Currently requires a plugin but will be fully integrated into future browsers

SVG Structure

<?xml version="1.0" standalone="no"?>

<!DOCTYPE svg PUBLIC "-//W3C//DTD SVG 20001102//EN"
"http://www.w3.org/TR/2000/CR-SVG-20001102/DTD/svg-20001102.dtd">
<svg width="200" height="200">
	<circle cx="100" cy="100" r="75" 
	style="fill:blue;stroke:green;stroke-width:6;" />

<?xml version="1.0" standalone="no"?>
<!DOCTYPE svg PUBLIC "-//W3C//DTD SVG 20001102//EN"
"http://www.w3.org/TR/2000/CR-SVG-20001102/DTD/svg-20001102.dtd">
<svg width="200" height="200">

	<text id="name" font-size="100%" x="24" y="118">KIEREN
	<animate attributeName="font-size" attributeType="XML" begin="0s"

	dur="7s" fill="freeze" from="0" to="40" />
	</text>

	<g>
	<circle	cx="100" cy="100" r="75" fill-opacity="1.0"
	stroke-width="6" style="fill:blue;stroke:green;">

	    <animate attributeName="cx" attributeType="XML"
	             begin="0s" dur="9s" fill="freeze" from="0" to="100" />

	    <animate attributeName="cy" attributeType="XML"
	             begin="0s" dur="9s" fill="freeze" from="0" to="100" />

	    <animate attributeName="r" attributeType="XML"
	             begin="0s" dur="7s" fill="freeze" from="0" to="75" />

	    <animate attributeName="fill-opacity" attributeType="XML"
	             begin="9s" dur="7s" fill="freeze" from="1" to="0.1" />

	    <animate attributeName="stroke-width" attributeType="XML"
	             begin="9s" dur="7s" fill="freeze" from="6" to="1" />

	</circle
	</g>
</svg>

Example

SVG supports:

Predefined shapes:

  • Rectangles
  • Circles
  • Ellipses
  • Polygons
  • Lines
  • Polylines
  • Paths/Bezier curves

SVG also supports:

  • Fill
  • Stroke
  • Text
  • Gradients
  • Masks
  • Opacity and visibility
  • Transformations
    • Rotation
    • Scale
    • Skew
    • Movement (translation)
  • Layers - position an SVG over a photograph

This means that complex images are possible

SVG authoring

  • By hand (can view source)
  • Adobe Illustrator
  • Corel Draw
  • Inkscape

Unfortunately no tool yet matches Macromedia Flash

Importantly, due to the open format, it is easy for CGI scripts (or developers) to produce SVG code "on the fly".

SVG integration

SVG can be integrated in to a wide variety of applications because of its XML structure. To (currently) incorporate an SVG image in an XHTML page you would use this code, in the future integration will be seamless:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
 "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">

<head>
<title>SVG Example</title>
</head>
<body>
<object data="example.svg" width="200" height="200" type="image/svg+xml">

<embed src="example.svg" width="200" height="200" type="image/svg+xml" />
</object>
</body>

</html>
