Navigational Tools

Ian Sealy

Introduction

  • Allow user to get around a Web site
  • Provide orientation to users
  • Easily distinguished from content
  • Function should be intuitive and obvious
  • Use standard navigational tools
  • What are standard navigational tools?

Standard Navigational Tools

  • Navigation bars or menus (global and local)
  • Mega drop-downs
  • Home button
  • Footer
  • Search
  • Breadcrumb trail
  • Tabs
  • Site map
  • Site index
  • Links!

Standard Navigational Tools

  • Not all navigational tools are suitable
  • Consider each when designing a site
  • Provide alternatives / duplicates
