Images and the Web

In preparing images for Web use there are a number of important considerations. These are:

  • Image file format
  • Image file size (in bytes)
  • Image file dimensions (in pixels)
  • Number of colours in the image (colour depth)

Image file formats

There are currently 3 image file formats supported by Web browsers. These are:

  • GIF (.gif) - Graphics Interchange Format
  • JPEG (.jpg or .jpeg) - Joint Photographic Experts Group
  • PNG (.png) - Portable Network Graphics

There are also a number of image formats which require a browser plugin. These include:

  • SWF (.swf) - Flash
  • SVG (.svg) - Scalable Vector Graphics

How images are saved

Computers save images in one of two ways:

  • As a 'grid' of picture elements (pixels) with 8 or 24 bits of information per pixel. Images saved in this way are called bitmap images. (.jpg, .gif, .png)
  • As a series of lines, curves and shapes through mathematical instructions. Images saved in this way are called vector-based images. (.swf, .svg)

Pixels and colour depth

GIF, JPEG and PNG are all bitmap image file formats. However, they differ in the amount of information (bits) used to save each of the pixels that make up the image. The more information available for each pixel the larger the number of colours available to the image; this is the colour depth.

Pixels and colour depth

  • GIF uses 8 bits (1 byte) to save each pixel. 8 bits per pixel gives a maximum of 256 colours.

Pixels and colour depth

  • JPEG and PNG use 24 bits (3 bytes) to save each pixel. 24-bit colour allows a maximum of 16.7 million colours.

File size (in bytes)

The size of an image file is determined by the number of pixels in the bitmap (the image dimensions) and the number of bits used to save each of those pixels (the colour depth). Therefore:

  • An 8-bit, 100x100 pixel image will be 10k.
  • A 24-bit, 100x100 pixel image will be 30k.

File size (in pixels)

The unit of measurement in Web design is the pixel. The pixel dimensions of an image will determine its size on screen. Resolution (points/pixels per inch) is irrelevant to a Web browser. A Web browser will display an image at its pixel dimensions whether it's 72ppi or 300ppi.

Monitor resolution

The pixel per inch resolution of most monitors is 72dpi. A rough guide to the pixel dimensions of monitors:

  • 14" monitor 640x480
  • 15" monitor 800x600
  • 17" monitor 832x624, 1024x768
  • 19" monitor 1024x768
  • 21" monitor 1280x1024
