WDD2

WDD2 aims to develop student skills in manipulating and controlling images for the Web. The module covers understanding media, graphics tools, screen design, Web versus other media, Web limitations and interface production using graphics tools. Students produce a portfolio of images and a banner animation for assessment.

Sessions:

  • WDD21
  • WDD22
  • WDD23
  • WDD24
