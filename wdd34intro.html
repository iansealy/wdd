Animation Formats

There are a number of ways of animating content in Web pages. These include:

  • GIF89a
  • Flash
  • HTML5/SVG/CSS3

HTML5

Supported by:

  • Compatibility chart

Tools:

  • HTML editors

HTML5

Advantages:

  • No plug-ins required
  • No special authoring software necessary
  • Can make use of the whole browser
  • Quick to download
  • Interactivity

HTML5

Disadvantages:

  • Labour intensive!

Flash

Supported by:

  • Netscape Navigator 2+
  • Internet Explorer 3+

Tools:

  • Adobe Flash

Flash

Advantages:

  • Streaming (short wait for animation to start)
  • High frame rate
  • Windowless
  • Supports transparency
  • Sound
  • Interactivity

Flash

Disadvantages:

  • Needs plug-in
  • Vector-based imagery is limiting
  • Accessibility problems

GIF89a

Supported by:

  • Netscape Navigator 2+
  • Internet Explorer 3+

Tools:

  • Adobe Photoshop
  • Adobe Fireworks
  • GIF Builder, GIFMation, Animation Shop

GIF89a

Advantages:

  • Simple to understand and create
  • Widely supported
  • Easy to integrate (no plug-ins)
  • Free or cheap authoring software
  • Supports transparency

GIF89a

Disadvantages:

  • File size limitations
  • Content limitations
  • Disjointed playback at different connection speeds
  • No sound
