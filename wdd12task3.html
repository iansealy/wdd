Task

  • Using tables for layout, create a home page
  • Can either create layout from scratch or reverse engineer an existing site
  • Include some navigational tools (e.g. a nav bar and a copyright page)
  • Linking to non-existent navigational tools is OK (e.g. a search tool)
  • Put comments in the HTML to describe each area of the page
  • The home page should allow a user to get to all the pages you've created so far on the course
  • Put the same look and feel on all the pages you've created so far
  • Navigation between each page should be easy
  • There should be no deadends
