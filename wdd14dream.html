Introducing Dreamweaver

Dreamweaver is one of the best WYSIWYG packages available. Why?

  • It's flexible and has a wide range of features
  • It's easy to use
  • It doesn't remove code that it does not recognise
  • You can easily access the HTML source
  • Integrated site management tools
  • Most importantly it's good at producing code that works in most browsers

However...

Dreamweaver CS4

Dreamweaver CS4 is the current version and is good at most things:

  • Can produce valid HTML
  • Good visual support for HTML and CSS using WebKit
  • Unpopular design view

Dreamweaver MX 2004 is installed on Birkbeck PCs and, although an older version, is still good.

What Can't It Do?

Some problems are still present:

  • Working with stylesheets generally less efficient than using a text editor
  • Hasn't gone as far as it could to banish font tags forever
  • It can still make changes you don't want
  • It can be "too helpful"
