Introducing Dreamweaver

Dreamweaver is one of the best WYSIWYG packages available. Why?

  • It's flexible and has a wide range of features
  • It's easy to use
  • It doesn't remove code that it does not recognise
  • You can easily access the HTML source
  • Integrated site management tools
  • Most importantly it's reasonably good at producing standards compliant code

Dreamweaver CC

Dreamweaver CC is the current version and is good at most things:

  • Good support for HTML5 and CSS3
  • Good visual support using Chromium's rendering engine
  • Integration with some PHP CMS tools
  • CSS inspector allows real time CSS editing/rendering
  • LiveView - browser-like behaviour within Dreamweaver
  • ... but, at a time when most designers are writing code, Dreamweaver is becoming more visual

How Much?

As students, you can get a discount but may need to apply before 29th November

What is available at Birkbeck?

Dreamweaver MX 2004 is installed on Birkbeck PCs and it's old (obviously given the name) and does have some problems:

  • Working with stylesheets far less efficient than using a text editor
  • Hasn't gone as far as it could to banish font tags forever
  • It can still make changes you don't want
  • It can be "too helpful"
