Animation Formats

There are a number of ways of animating content in Web pages. These include:

  • Dynamic HTML (DHTML)
  • Flash
  • GIF89a

Dynamic HTML

Supported by:

  • Netscape Navigator 4+
  • Internet Explorer 4+

Tools:

  • Macromedia Dreamweaver
  • HTML editors

Dynamic HTML

Advantages:

  • No plug-ins required
  • No special authoring software necessary
  • Can make use of the whole browser
  • Quick to download
  • Sound
  • Interactivity

Dynamic HTML

Disadvantages:

  • Poor cross-platform compatibility
  • Slow to re-draw screen therefore slow frame rate

Flash

Supported by:

  • Netscape Navigator 2+
  • Internet Explorer 3+

Tools:

  • Macromedia Flash
  • Graphics applications (Freehand, Illustrator)

Flash

Advantages:

  • Streaming (short wait for animation to start)
  • High frame rate
  • Windowless
  • Supports transparency
  • Sound
  • Interactivity

Flash

Disadvantages:

  • Needs plug-in
  • Vector-based imagery is limiting

Flash example

GIF89a

Supported by:

  • Netscape Navigator 2+
  • Internet Explorer 3+

Tools:

  • Adobe ImageReady
  • Macromedia Fireworks
  • GIF Builder, GIFMation, Animation Shop

GIF89a

Advantages:

  • Simple to understand and create
  • Widely supported
  • Easy to integrate (no plug-ins)
  • Free or cheap authoring software
  • Supports transparency

GIF89a

Disadvantages:

  • File size limitations
  • Content limitations
  • Disjointed playback at different connection speeds
  • No sound
