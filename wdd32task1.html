Task

Use Photoshop to create a mock-up of your module 3 assignment digital portfolio.

  • Create a new 1020x1020 pixel document
  • Open an existing image and make a square selection from it, then paste this selection into your main image. Repeat this with a number of other images
  • Use a grid to help you create the layout
  • Try moving the images around the grid and grouping them in different ways
  • Add a larger version of one of the images
  • Download a grid here
