IMG

To include an image in a Web page you need to use <img src="URL">. IMG elements are inline. Therefore, if the tag is added to a line of text the image will appear in the line of text at the point where the tag was inserted.

Attributes to IMG tag

There are a number of attributes which can be added to the IMG tag. These are:

  • alt="textmessage"
  • height="n" width="n"

Images as hyperlinks

It is possible to turn an image into a hyperlink by surrounding the image tag with the <a href="URL"> </a> tags.
