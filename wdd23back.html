Background Images

It is possible in HTML to put in an image as the background of a Web page. To do this add the attribute background="url" to the body tag.

Example:

<body background="backimage.gif">

Background Images

If the image is large enough it will fill the whole of the background of the browser window. If it is smaller than the size of the browser window it will be tiled. Therefore, it is best to:

  • Create an image that will fill the background window of the largest monitor

or

  • Create an image that will repeat succesfully

CSS

It is also possible to insert a background image with CSS using the background-image: url(url) property. The background properties also give more control over the image placement.

CSS

  • body { background-image: url(image.gif);}
    Repeats image to fill the background
  • body { background-image: url(image.gif); background-repeat: repeat-x }
    Repeats image horizontally in a single row
  • body { background-image: url(image.gif); background-repeat: repeat-y }
    Repeats image vertically in a single column
  • body { background-image: url(image.gif); background-repeat: no-repeat }
    Does not repeat image

CSS

  • body { background-image: url(image.gif); background-attachment: fixed }
    Background image does not scroll with the page's content
  • body { background-image: url(image.gif); background-attachment: scroll }
    Background image scrolls with the page's content

CSS

To position a background image use the background-position property. Values for this property come in pairs and specify the distance from the left and top edges of the browser window. This distance can be specified as:

  • A percentage of the browser window
    body { background-image: url(image.gif); background-position: 50% 50% }
  • A pixel measurement
    body { background-image: url(image.gif); background-position: 100px 100px }
  • A keyword (top, bottom, left, right, center)
    body { background-image: url(image.gif); background-position: center center }
