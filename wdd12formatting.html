Introduction

  • Ideally done using CSS (cascading stylesheets)
  • Originally HTML only allowed headings, bold, italics, etc...
  • In 1995 Netscape 1.1 introduced the <font> tag
  • Only use <font> tag until we cover CSS
  • The only use for font tags is HTML emails

Face Attribute

  • Introduced by Microsoft in August 1995 (and in Netscape 3.0)
  • <font face="Arial">
  • <font face="Times New Roman">
  • <font face="Symbol">
  • <font face="serif">
