Mailing Lists

Two mailing lists have been set up to support this course.

If you need to contact the course tutors then you should send a message to our mailing list:

  • wddtutors@bbkweb.org

The wddtutors@bbkweb.org mailing list is a private list which only the course tutors can read.

There is also a mailing list which all students must join. We will use this list to contact you, but you should also use it as a discussion forum to help each other with problems you may be having and to share details of good resources you have found.

To join the list go to the list's Web page and subscribe. The membership of the list is restricted and all new members must be authorised by the course tutors.

To send messages to the list, you should email:

  • wdd0607@bbkweb.org
