Task

Reverse engineer a mockup in Photoshop.

  • Download this screenshot of a Web page as a base
  • Open the file in Photoshop
  • Draw shapes on to the base image using the rectangle tool
  • Try changing the colour of the shapes. You can use a swatch downloaded from Kuler or Colour Schemer.
  • Try and replicate some of the text on the screenshot. You can get dummy text here (opens in new window).
