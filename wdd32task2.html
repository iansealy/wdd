Task

Mock-up your module 3 assignment image/documentation page.

  • Use a grid you've downloaded or my example grid as a base
  • Add large and small (before and after) versions of an image
  • Add 2 or 3 paragraphs of text description. You can get dummy text here (opens in new window).
  • Attach a list of optimisation details to the large version of the image. The details could include the following:
    1. File format
    2. Quality setting
    3. Original file size
    4. Optimised file size
    5. Quality setting
    6. Number of colours
