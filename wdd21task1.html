Task

  • Collect some images from a variety of Web sites. Try to get some photographs, some illustrations and some graphics.
  • Make a note of the image type (e.g. photograph, illustration, etc...) and the file extension (.gif, .jpg, or .png).
  • Make a note of the file size.
  • Open the images in Photoshop, note the file size again and the image dimensions.
  • If the image is a GIF then also have a look at the colour table and make a note of the number of colours.
