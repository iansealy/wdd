Including images in a Web page

To include an image in a Web page you need to use <img src="url" />. img elements are inline. Therefore, if the tag is added to a line of text the image will appear in the line of text at the point where the tag was inserted.

Example:

This is a line of text <img src="birkbeck.gif" /> with an image in it.

This is a line of text with an image in it.

Offering alternate text

The "alt" attribute is required for all img tags in html and xhtml. The alternate text will appear in place of an image that fails to load in a browser that is displaying images or read to someone using screenreading software. There is a guide to using alternate text here.

Example:

This is a line of text <img src="birkbeck.jpg" width="200" height="63" alt="Birkbeck, University of London" /> with an image in it.

Width and Height

Supplying the browser with an image's width and height will allow it to load content around the image as the image loads. You can do this with the width and height attributes.

Example:

This is a line of text <img src="birkbeck.jpg" width="200" height="63" alt="Birkbeck, University of London" /> with an image in it.

Other attributes to img tag

There are a number of attributes which can be added to the IMG tag.

Images as hyperlinks

It is possible to turn an image into a hyperlink by surrounding the image tag with the <a href="url"> </a> tags.

Example:

<a href="http://www.bbk.ac.uk"><img src="birkbeck.gif" width="200" height="63" alt="Birkbeck Logo" /></a>
