WDD31 Reading List

1. Websites

  • Nathan Shedroff - Lots of "experience design" info.
  • Boxes and arrows - Forum for latest thinking in Information Architecture
  • JJG.net - More IA resources.
  • Usableweb - Fantastic usability portal.
  • UseIt - Jakob Nielsen's influential musings (and self-promotion!)
  • IBM Ease of use - Lots of user articles and resources.
  • IAWiki - Currently down due to a hardware problem, but should reappear soon.
  • Usability News - Latest UK news, events and jobs.

2. Books

Beyer, H. & Holtzblatt, K. (1998) Contextual design. Morgan Kauffman.
Influential (and seriously detailed) book on contextual enquiry.

Brown, D (2006) Communicating Design: Developing Web Site Documentation for Design and Planning. New Riders.

Cooper, A. (1999) The inmates are running the asylum. Sams.
The inventor of "personas" argues the case for user-centred design in software development.

Garrett, J.J. (2002) The Elements of User Experience. New Riders.

Hackos, J. & Redish, J. (1998) User interface task analysis. John Wiley & Sons.
THE book for task analysis and interface design, but only really for the truly dedicated!

Krug, S. (2005)Don't make me think! Que.
Short, easy-to-read and informative overview of web usability. A bit flippant about user testing at the end of the book, though.

Kuniavsky, M. (2003) Observing the User Experience. Morgan Kaufman.

Moggridge, B. (2006) Designing Interactions. MIT Press.

Mulder, S. & Yaar, Z. (2006) The User Is Always Right: A Practical Guide to Creating and Using Personas for the Web. New Riders.

Nielsen, J (2006). Prioritizing Web Usability. New Riders.

Nielsen, J (2000). Designing web usability: the practive of simplicity. New Riders.
It's Jakob. It's a classic.

Norman, D.A. (2005) Emotional Design: Why We Love (or Hate) Everyday Things. Basic Books.

Norman, D.A. (1998) The design of everyday things. MIT Press.
Great insight into how understanding human psychology can greatly affect the way we design things.

Raskin, J. (2000) The humane interface. Addison Wesley.
A glimpse into the future of computer interfaces?

Rosenfeld, L. & Morville, P. (2002) Information architecture for the world wide web. Second Edition. O'Reilly.
The classic book on IA. Affectionately known as "The Polar Bear Book". Make sure that you get the second edition - it's way more comprehensive and up-to-date than the first.

Reiss, E. (2000) Practical information architecture. Addison Wesley.
Concise overview of IA.

Saffer, D. (2006) Designing for Interaction: Creating Smart Applications and Clever Devices. New Riders

Veen, J. (2001) The art & science of web design. New Riders.
Great overview of web design. Includes illuminating chapters on user behaviour and site architecture.

Wodtke, C. (2002) Information Architecture: Blueprints for the Web. New Riders.
