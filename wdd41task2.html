Task

  • Read the tutorials
  • Create a feedback form for your own Web site
  • Your form should collect the user's name, email address, location and comments
  • Use DreamHost's FormMail script to get the results of each form submission emailed to you
  • Note: You'll need to ask for an @bbkweb.org email address
  • Once you've mastered that, use a remotely hosted form processing script to get the results of each form submission emailed to you
  • If you're feeling really clever, try installing and using the nms version of FormMail
