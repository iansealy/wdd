Task

  • Create two directories in your Web space, one called "yin" and the other called "yang"
  • Put an index.html (containing some arbitrary text) in the "yin" directory using FileZilla
  • Put some random files (not index.html) in the "yang" directory using FileZilla
  • Access these directories via the Web (i.e. via a URL starting with http://)
