Image Replacement

A number of techniques have been developed for replacing text with images in a way that separates the presentational or decorative from the content of the document using CSS.

Example

text-indent

Using a combination of background-image and text-indent properties it is possible to place text out of view leaving just the image as the background to the element.

Example:

<h1>NYPL Online Style Guide</h1>

h1 {
height: 45px;
background-image: url(styleguidehead.gif);
background-repeat: no-repeat;
text-indent:-1000em;
}

Problems with text-indent

The disadvantage of this method is that if images are turned off in the browser but CSS is on then there will be no image and no text, just an empty space.

text-indent and span

This method hides the text behind the image so that when images are off and css is on the text is visible. The background image is on the empty span tags inside the h1.

Example:

<h1><span></span>NYPL Online Style Guide</h1>

h1 {
position: relative;
width: 425px;
height: 50px;
overflow: hidden;
}

h1 span {
position: absolute;
left: 0;
top: 0;
width: 100%;
height: 100%;
background-image: url(styleguidehead.gif);
background-repeat: no-repeat;
