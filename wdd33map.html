Image Replacement

A number of techniques have been developed for replacing text with images in a way that separates the presentational or decorative from the content of the document using CSS.

Example

Image Replacement

Using a combination the font, text-shadow and color properties it's possible to 'hide' text so that a background image beneath is visible.

Example:

font: 0/0 a;
text-shadow: none;
color: transparent;

Image Replacement

In this example:

  • The shorthand font property values 0/0 set the size and line height of the text to zero
  • The a is included to stand in for the mandatory font-family value (the presumption is that no one will have a font called a installed)
  • The text-shadow property value none ensures that no shadow is applied to text
  • The color property value transparent sets the text to transparent
