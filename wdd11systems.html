Introduction

  • Everyone has a Birkbeck username and password for accessing computers in Birkbeck
  • You can (and should) change your password
  • If you forget your username then a list is available

Network Drives at Birkbeck

  • Network drive - N:
  • Can be used for storing files you are currently working on
  • Available on any Birkbeck computer you login on
  • You are limited to 1 GB of data

File Extensions

  • By default Windows hides file extensions
  • football.html and football.htm are different files, but will appear to be the same
  • Need to show all file extensions
  • Done by:
    Windows Explorer --> Tools --> Folder Options --> View tab --> Uncheck "Hide extensions for known file types"
  • Or:
    Windows Explorer --> Organize --> Folder and search options --> View tab --> Uncheck "Hide extensions for known file types"
  • You'll want to make this change on your home computer as well as at Birkbeck

Web Space

  • Each student has their own Web space on a DreamHost server
  • Using professional hosting so you can experience real systems
  • A list of URLs is available
  • For example, my username is iansealy and my Web space is at http://iansealy.wdd1617.bbkweb.org/

Different Password

  • Username is same for Birkbeck and for DreamHost
  • Passwords are different; changing your Birkbeck password will not change your DreamHost password (and vice versa)
  • DreamHost password is changed via a more complicated method

Changing DreamHost Password

  • Run PuTTY
  • In the Host Name box type bbkweb.org and press the Open button
  • When the "login as:" prompt appears, type your username and press Enter
  • Then enter your password and press Enter
  • (Note that you won't get any indication you're typing your password)
  • Then type passwd and then enter your existing password and press Enter
  • Then enter your new password (twice)
  • Finally type exit and press Enter

FTP

  • Can access DreamHost Web space using FTP (File Transfer Protocol)
  • Allows transfer of files from your local computer (at home or here at Birkbeck) to a remote computer (and vice versa)

Basic Scenario

  • All FTP programs are different, but basic scenario is:
    • Specify computer to connect to
    • Give username and password
    • Specify files to GET from the remote computer or
      Specify files on your computer that you want to PUT on the remote computer
  • The computer you need to FTP to to get at your Birkbeck Web space is called:
    bbkweb.org
  • Web pages need to go in the htdocs directory

Recommended FTP program

  • The FTP client we recommend and use at Birkbeck is FileZilla
  • FileZilla demo

FTP

  • Create directory called test in htdocs subdirectory of your Web space
  • Go to this directory on the Web by appending test to your personal URL
  • For example, my URL is http://iansealy.wdd1617.bbkweb.org/
  • Create a text file in your N: drive called yourname.txt and put some text in it
  • FTP this text file to your Web space and put it in the new test directory
  • View via the Web

Index Documents

  • View new test directory on Web
  • Gives directory listing (including new yourname.txt file)
  • Create a file called index.html in Notepad and save it to the N: drive and FTP it to your test directory
  • If a Web browser requests a directory then an index document is sent back to the browser
  • If no index document is present then a directory listing is sent back to the browser OR an error message is displayed
  • Former is the case for DreamHost Web server
  • index.html is most common name for an index document, but others are possible, e.g. index.htm, welcome.html, home.html, Default.htm
